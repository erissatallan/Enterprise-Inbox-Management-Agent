# Setup Manifests and Templates

This folder contains importable manifests and setup templates used by `docs/INSTALLATION_TUTORIAL.md`.

## Files

- `slack-app-manifest.yaml`
  - Import this directly into Slack App creation (`From a manifest`).

- `google-gmail-oauth.template.json`
  - OAuth scope template and environment mapping for Gmail integration.

- `microsoft-graph-mail.template.json`
  - Planned Outlook/Graph app registration template.

- `microsoft-teams-bot.template.json`
  - Planned Teams bot setup template.

- `business-handoff-webhooks.template.env`
  - Environment template for Jira/Linear + HubSpot/Salesforce webhook handoff.
